Comer En Morelos
=================

Comer En Morelos es una aplicación movil de código abierto hecha usando PhoneGap (Apache Cordova). Esta hecha enteramente usando HTML, CSS, y JavaScript, y corre en numerosas platformas. 

Check out a video of it in action at: 
* http://youtu.be/QjDD1l_37B8

Learn how it was built at: 
* http://www.tricedesigns.com/?p=2156

Comer En Morelos usa:
* PhoneGap: http://www.phonegap.com
* App-UI: http://triceam.github.com/app-UI/
* Mustache: https://github.com/janl/mustache.js
* jQuery: http://jquery.com/
